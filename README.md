Kranus Health – Landing Page (Concept Prototype)

This repository contains a landing page concept and prototype created for Kranus Health, a healthtech company focused on digital therapies in urology.

🌐 Live Links

Current Website: https://www.kranushealth.com/de/home

Prototype (this project): https://simran2809.github.io/Kranus-Health/

📌 About Kranus Health

Kranus Health is a digital health company building evidence-based therapies for urological conditions such as erectile dysfunction, incontinence, and bladder disorders. Their solutions combine medical expertise with technology to improve access to care and patient outcomes.

🎯 Project Goal

The goal of this project was to:

Reimagine the landing page experience

Improve clarity, storytelling, and engagement

Create a modern, conversion-focused UI

Explore creative directions using rapid prototyping tools

⚡ What I Built

A fully functional static landing page prototype

Clean and structured information hierarchy

Improved visual storytelling and UX flow

Designed for fast loading and responsiveness

🎨 Creative Approach

Built using AI-assisted development (Claude + vibe coding)

Focused on:

Emotional connection (health + confidence)

Clear value proposition

Modern healthtech aesthetics
